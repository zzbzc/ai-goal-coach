# 设计系统 — AI 目标教练

## 产品背景
- **产品是什么：** 基于 AI 的目标追踪与教练应用
- **目标用户：** 需要目标管理和进度追踪的个人用户
- **领域/行业：** 生产力工具 / 个人成长
- **项目类型：** Flutter 移动应用

## 美学方向
- **方向：** 简洁专业
- **装饰程度：** 克制 — 装饰服务于功能，柔和阴影创造层级，无多余图形元素
- **氛围：** 清爽、可信赖、数据驱动。类似主流 SaaS 应用的视觉语言，强调专业感和信任感

## 配色方案
- **方案：** 自然亲和 — 森林绿主色 + 暖灰中性色 + 珊瑚红点缀
- **主色：** Forest Green `#2D5A46` — 用于主按钮、激活状态、关键数据点
- **主色阶调：**
  - 50: `#E8F5EF` (浅色背景)
  - 100: `#D1E8DC` (浅色边框)
  - 200: `#A3D1B9` (浅色强调)
  - 500: `#2D5A46` (主色)
  - 700: `#1F3D30` (深色强调)
  - 900: `#0F1F18` (深色文字)
- **辅色：** Sage Green `#87A892` — 用于次要操作、进度指示
- **点缀色：** Coral `#E07A5F` — 用于关键操作、成就徽章、情感表达
- **中性色：** 暖灰系 — 带微妙黄/棕倾向
  - 50: `#FAF9F7` (主背景)
  - 100: `#F2F0EB` (次级背景)
  - 200: `#E6E2D8` (边框)
  - 400: `#B5ADA0` (弱化的文字)
  - 600: `#7D7265` (次要文字)
  - 900: `#3D342B` (主要文字)
- **语义色：**
  - 成功 `#4A7C59` (森林绿系)
  - 警告 `#D4A373` (琥珀/陶土)
  - 错误 `#C96859` (珊瑚红系)
  - 信息 `#6B9AC4` (灰蓝色)
- **深色模式：** 表面色加深，主色降低饱和度 15%，点缀色提高明度

## 字体排印
- **UI/标签：** Plus Jakarta Sans — 现代、几何感强，适合移动端小字号显示
- **正文：** DM Sans — 高可读性，开放字怀，适合长文本阅读
- **数据/表格：** JetBrains Mono — 支持 tabular-nums，等宽数字对齐
- **字阶：** 12/14/16/18/20/24/32px（1.25 比例）

### 字体加载
```html
<link href="https://fonts.googleapis.com/css2?family=Plus+Jakarta+Sans:wght@400;500;600;700&family=DM+Sans:opsz,wght@9..40,400;500;700&family=JetBrains+Mono:wght@400;500&display=swap" rel="stylesheet">
```

### 字号规范
| 级别 | 字号 | 字重 | 用途 |
|------|------|------|------|
| 32px | Bold (700) | 主标题/页面标题 |
| 24px | SemiBold (600) | 章节标题 |
| 20px | Medium (500) | 卡片标题 |
| 16px | Regular (400) | 正文 |
| 14px | Regular (400) | 次要文字/说明 |
| 12px | Medium (500) | 标签/元数据 |

## 间距系统
- **基准单位：** 8px
- **密度：** 舒适型 — 适合移动端拇指操作
- **阶梯：** 4/8/12/16/24/32/48/64px

### 间距使用规范
| 名称 | 值 | 用途 |
|------|-----|------|
| 2xs | 4px | 图标与文字的微小间距 |
| xs | 8px | 紧密元素间距 |
| sm | 12px | 表单元素内边距 |
| md | 16px | 卡片内边距、标准间距 |
| lg | 24px | 区块间距 |
| xl | 32px | 大区块间距 |
| 2xl | 48px | 页面级间距 |
| 3xl | 64px | 超大间距 |

## 布局
- **方案：** 移动优先网格 — 底部导航 + 卡片式布局 + 清晰的信息层级
- **网格：** 响应式列数（移动端 1 列，平板 2 列，桌面 3-4 列）
- **最大内容宽度：** 1200px
- **圆角：** 克制的圆角 — sm: 4px, md: 8px, lg: 12px, full: 9999px

### 圆角使用规范
| 名称 | 值 | 用途 |
|------|-----|------|
| sm | 4px | 小型元素、标签 |
| md | 8px | 按钮、卡片、输入框 |
| lg | 12px | 大卡片、模态框 |
| full | 9999px | 头像、开关、徽章 |

## 动效
- **方案：** 功能优先 — 仅必要的状态过渡，辅助理解
- **缓动：** 进入 ease-out / 离开 ease-in / 移动 ease-in-out
- **时长：** 微交互 150ms / 短 250ms / 中 400ms

### 动效使用规范
| 场景 | 时长 | 缓动 |
|------|------|------|
| 按钮悬停 | 150ms | ease-out |
| 页面切换 | 250ms | ease-in-out |
| 展开/收起 | 400ms | ease-in-out |

## 设计决策日志
| 日期 | 决策 | 理由 |
|------|------|------|
