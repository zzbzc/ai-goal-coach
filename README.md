# AI Goal Coach

一个使用 Flutter 构建的 AI 目标教练应用。

## 项目简介

AI Goal Coach 是一款智能目标管理应用，帮助用户设定、跟踪和实现个人目标。

## 技术栈

- **框架**: Flutter
- **支持平台**: Android, iOS, Web, macOS, Linux, Windows

## 开始使用

### 环境要求

- Flutter SDK >= 3.0
- Dart SDK

### 安装依赖

```bash
flutter pub get
```

### 运行应用

```bash
# 运行在 Chrome 上
flutter run -d chrome

# 运行在 Android 设备上
flutter run

# 运行在 iOS 模拟器上
flutter run -d ios
```

### 构建发布版本

```bash
# Web
flutter build web

# Android APK
flutter build apk

# iOS
flutter build ios
```
